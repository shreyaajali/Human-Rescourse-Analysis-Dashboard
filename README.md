# Human-Rescourse-Analysis-Dashboard
This project contains an interactive Human Rescource Analysis Dashboard built in Power BI. It provides clear insights into employee demographics salary distribution, job roles qualifications, leave balance, and staff growth. The visuals help HR teams quickly understand workforce trends and make data-driven decisions.
Includes:
*Gender breadown
*Key HR KPIs (headcount, avg salary, avg leave balance)
*Qualification vs salary
*Job title distribution & top earners
*Staff growth over time
*Leave balance analysis
